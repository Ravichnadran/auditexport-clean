# Changelog

All notable changes to AuditExport are documented here.

---

## v0.2.0

### Added
- SOC 2 change management correlation (PR → CI → merge)
- CI/CD execution evidence (GitHub Actions)
- Failed CI negative evidence
- Reviewer independence checks
- SOC 2 control mapping (CC6, CC7, CC8)
- Audit window filtering (`--from-date`, `--to-date`)
- Auditor-readable SOC 2 assertions

### Improved
- Deterministic evidence generation
- Explicit handling of unavailable controls

---

## v0.1.0

- Initial ISO/IEC 27001 evidence support
- GitHub governance and access controls
- Hash-verified evidence packaging
